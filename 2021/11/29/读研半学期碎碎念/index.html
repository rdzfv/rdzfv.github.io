读研半学期碎碎念

Author: xyy Date: 十一月 29, 2021  21:36:06 Category: 个人碎碎念

​ 现在是新版博客！自从11月初服务器到期，就一直关站了，一直没时间去重新搭（太忙了太忙了）。

新版本的博客搭建

​ 于是今天在一门摸鱼课开始《博客复活计划》。原来的写博客体验并不是很好，我要先在本地的markdown编辑器里写好内容，再连上服务器新建新的博文 hexo new article_title 并把内容粘上去，用 hexo -g 生成可被浏览器解析的前端页面，再通过 hexo -d 上传至 github 仓库。这样的问题是，每次我想法发布博客，都需要连接服务器，并进行内容的粘贴。这对于一个究极懒癌患者程序员来说是难以忍受的（好吧，也这样用了一两年了）。

​ 于是，我在想，我能不能在本地编写博客，通过hexo的 hexo -d 命令将生成的前端内容同步到 github 仓库。再由远程服务器定时地进行拉取，获取最新的博客内容。于是便诞生了如下图所示的结构。所以现在我就可以愉快地在本地写博客，然后静静地等待 www.xuyuyan.cn 的更新啦！

（此处鸣谢胡大橘同学 https://www.huhaorui.com/ 提供的部分 debug 帮助）


开始念叨最近的读研生活

1. 对短期内的生活焦绿

​ 读研的生活真的真的真的比本科要劳累得多。虽然在课程上的安排并不算紧，可能也就是类似本科生大一第一学期的课程量吧，并且通常没有课后作业（结课作业除外），但却真真切切地感受到精神上的压力。

​ 因为课后作业不多，所以有限的课后时间几乎全是自己支配。

  • 不像是本科写完作业就可以说结束了今天的”任务“，有很大的可能性你今天除了上课，没有”任务“，你需要自己找事情给自己干。每当自己一闲下来，一想到自己距离毕业还遥遥无期，就感觉非常有压力。
  • 并且，因为自己也是小白，你需要做好花很多时间做无用功的心理准备。比如当前学习的内容可能不一定是以后自己用于毕业设计或者小论文的方向之类。
  • 自己的数学功底很弱，虽然最近的无网课空档期在看小竺老师给的书，但是基本只能做到 了解 ，比较难去 理解。再想到以后写小论文，做一些改进，肯定是需要一些数学能力的，就蛮焦绿的。
  • 出于一些奇奇怪怪的原因，在学日语（因为上海疫情的原因，今年12月5日的能力考取消了，一时不知道是欢喜还是失落）。每个周末基本上花上一些时间看教材、做一些语法笔记，平常有时也会看看单词。但有时也会质问自己：这真的对你的毕业有帮助吗？
  • 基于以上，就很少舍得给自己额外的摸鱼事情花时间，比如搭博客，尝试一些奇奇怪怪但又觉得有意思技术小工具等。（当然还是会抽空打一打游戏，还是得劳逸结合！）

2. 对长期的择业焦绿

​ 还要做技术吗？似乎我已经问过自己好多次了。从大一转专业的时候的热血沸腾、雄心壮志，变成了对自己的一次次质疑。其实也不是我自己妄自菲薄，只是我觉得人应该面对现实，而不是沉浸在”我今天写完作业了没“。

​ 本科认识很多大佬朋友，精弘后花园的群友应该大厂率80%以上吧，后花园还在读研的几个同学也都是大佬。就拿tx的胡某橘来说吧，因为某种特殊原因，和他的技术交流比较多，真真切切地感受到自己和大佬的差距。

  • 自身习惯问题：每次用到新的工具、新的命令，往往没有去弄懂其内在原理的意识，总是追求能用、会用、跑通，完成任务。久而久之，知识的深度就和大佬有了鸿沟。
  • 对技术的其实也没有太大的热爱：我身边的大佬们的知乎推荐一半都是奇奇怪怪的技术讨论，（另一半华强北）。有事没事就去刷刷文章，或者去公司的内网论坛看巨佬发言。而我自己真正认定为休闲娱乐的时候，还是会选择追星、看猫猫、打游戏等，如果是看一些技术博客之类的，一般就认定为学习了。久而久之，知识的广度就和大佬有了鸿沟。
  • 想要的太多，能力不够：在自身能力比较有限的情况下，又想有还不错的课业成绩、又想有奇奇怪怪的个人业余兴趣方向技能点、又想在自己目前从事的事情上可以做的不错、又想和身边其他大佬一样具有广博的知识……其实人生就是选择题吧，你总是需要选择自己当前需要做什么事情。到底你的选择是不是正确呢，是不是适合你自己呢？谁知道呢。这大概也就是，为什么努力的人那么多，成功的人却总是极少。

希望大家在面对工作、考研、调剂等重大决定的时候，务必务必要再三权衡！
